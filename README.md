#NodeJS FoodCart
### Techniques
----
Backend: NodeJS, Express, Express-Session, Mongoose, MongoDB.<br>
Documentation: Swagger.<br>
Test: Postman.
###Run Locally
----
Install packages and dependencies:<br>
`> npm install `<br>

Run database:<br>
`> mongod`

Seeding Data to Database:<br>
`> node ./seed/seed-product`


Run app:<br>
`> node app.js`
